# 割り箸カード v49 試合後フロー版

## 変更内容

- オンライン試合終了後に3つの行動を追加
  - 同じデッキでもう一戦
  - デッキを変更して再戦
  - ロビーへ戻る
- 「同じデッキでもう一戦」は両者が選択した時だけ、新しい matchId で再戦を自動開始
- 片方でも「デッキを変更して再戦」を選ぶと、両者の準備状態を解除
  - 選択した本人はデッキ編集へ移動
  - 相手は同じ部屋のロビーで待機
  - 編集後はロビーに戻り、再度準備完了して新デッキを提出
- 片方でも「ロビーへ戻る」を選ぶと、再戦要求を終了して両者を同じ部屋の待機状態へ戻す
- 試合後の選択状況を Firebase の `postMatch` で同期
- 結果画面を閉じても、オンライン試合終了後は盤面上部の「結果を見る」から再表示可能
- オンライン対戦中はCPU戦専用の「同じ難易度で再戦」ボタンを非表示
- `game.js?v=49` / `style.css?v=49` に更新

## 試合後同期の考え方

- hostChoice / guestChoice を別々に保存
- 両者 rematch の時だけ即再戦
- deck または lobby が選ばれた場合はホストが共通の resolvedAction を確定
- resolutionId により同じ画面遷移を二重適用しない

## テスト推奨

1. 両者が「同じデッキでもう一戦」→新しい試合が両端末で開始
2. ゲストだけ「デッキを変更して再戦」→ゲストは編集画面、ホストはロビー待機
3. 編集後にゲストがロビーへ戻り、両者準備完了→変更後デッキで開始
4. 片方が「ロビーへ戻る」→両者が待機状態へ戻る
5. 結果画面を「盤面を見る」で閉じる→「結果を見る」で再表示
