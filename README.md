# 割り箸カード v46 同期演出・罠秘匿版

## 変更内容
- Firebase の盤面状態とは別に `match.fx` を追加し、演出イベントだけを同期
- 通常カード使用時の中央ポップアップを相手側にも再生
- 通常攻撃の「攻撃した手 → 攻撃対象」のアニメーションを相手側にも再生
- 「分ける」の中央ポップアップを相手側にも再生
- 罠が実際に発動した瞬間の公開ポップアップを相手側にも再生
- 伏せ罠の設置ログからカード名を削除し、相手への情報漏洩を防止
- 演出同期は `match.state` を変更しないため、盤面状態の巻き戻りを起こしにくい構造

## 注意
- 演出イベントは通信状況によってわずかに時間差が出る場合があります。
- 伏せ罠は設置中は名前を共有ログに出さず、発動時にのみ公開します。
