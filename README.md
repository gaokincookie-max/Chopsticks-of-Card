## 主な仕様

### 雷撃
雷撃が乗った攻撃では、力学発電・バイオティックEなど
「その攻撃で与えた本数を参照して充電を得る効果」は発動しない。

太陽光発電・化学発電など、ダメージと無関係な充電獲得は通常どおり。

### E=mc²
最後の手が0になって敗北する時、条件を満たせば手札から自動発動。

- 通常攻撃・カード効果: 充電6以上
- ロジックアトリエ: 充電10
- 充電を全消費
- 最後の手を4にする
- 専用ポップアップあり

現版では通信待ちによる停止を避けるため、発動条件を満たした場合は自動発動。

### 廉価バッテリー
次の自分ターンから2回、開始時に充電-2。
専用ポップアップで残り回数を表示。

### エネルギーバリア
次の自分ターン開始時まで、受ける本数-2。最低0。

### 発電系加護
- 力学発電: 攻撃で与えた本数と同じ充電
- バイオティックE: 相手の手を0にした時、攻撃本数×2
- 化学発電: 手札からカードを使用するたび充電1
- 太陽光発電: 自分ターン開始時に充電2
