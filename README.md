# 割り箸カード v48 PVPデッキ保存・コード対応版

GitHub Pages でそのまま動かせる分割版です。

## v48の主な変更

- PVP用デッキにデッキコード機能を追加
  - コード作成
  - コード読込
  - コピー
- PVP用デッキに保存スロットを追加
  - スロット1〜3
  - 保存/読込
- オンライン対戦開始時、ホスト/ゲストがそれぞれ自分のPVPデッキをFirestoreに保存
- 試合開始時に、ホストはホストのデッキ、ゲストはゲストのデッキを使うように修正
- GitHub Pagesのキャッシュ対策として `game.js?v=48` / `style.css?v=48` を使用

## 重要

v47以前は、PVP試合開始時にホスト側デッキが両者に使われる可能性がありました。
v48では `rooms/{roomId}` に `hostDeck` / `guestDeck` を保存して、それぞれ別デッキを使います。
