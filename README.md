# 割り箸カード v80 光速回路・特別演出版

## 追加
光速回路が正常発動した時、専用のシネマティック演出を追加。

## 演出内容
- メイン表記: `OVERCLOCK`
- サブ表記: `〇〇の光速回路起動 / 充電カード使用制限解除`
- 紫ベースのびりびりした電撃風エフェクト
- 画面全体に走る紫のスパークと明滅
- フィナーレやロジックアトリエと同じ special-fx-layer を利用

## 発動条件
- 光速回路が充電Lv.10で正常発動した時のみ
- 充電不足で不発だった場合は演出なし

## オンライン
- 対人戦では両プレイヤーに同じ OVERCLOCK 演出を表示
- `lightSpeedCircuit` FX を追加して同期

## 既存仕様
- そのターン、充電カードを何枚でも使用可能
- 充電カードの終端を無視
- ターン終了時に充電0
- 反動は次の自分ターンに発生
