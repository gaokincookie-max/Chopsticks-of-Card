# 割り箸カード v61 PVP共通カード基盤版

## 主な変更

- PVP専用の `FRIEND_SIMPLE_LIBRARY` を削除
- PVPのカード名・コスト・説明はCPU戦と同じ `CARD_LIBRARY` を正本として使用
- PVPデッキ編集には、現在PVPで動作確認可能なCPU戦カードだけを表示
- v60以前の旧PVP専用カードIDを正式なCPU戦カードIDへ自動移行
- 未対応カードには簡易版効果を作らず、PVPデッキ候補から除外
- カード使用ポップアップのログ検出を修正（ログ30件上限で先頭が押し出された場合にも対応）

## 現在の方針

カード情報は共通化済みですが、PVPの効果実行部分にはまだ専用処理が残っています。今後は対応カードから順にCPU戦と同じ処理・操作フローへ統合します。
