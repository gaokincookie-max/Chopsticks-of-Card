# 割り箸カード v53 特別演出強化版

## 変更内容

- フィナーレの暗転、タイトル、光の輪、明転、着弾のテンポを全体的にゆっくり調整
- フィナーレの「溜め」を強くし、必殺技らしい見せ方へ変更
- ロジックアトリエに専用演出を追加
  - 短いロック表示
  - 高速の斜線・閃光
  - 対象の手を粉砕する専用アニメーション
  - 対象を0にする処理と演出を接続
- ロジックアトリエ専用演出をオンラインの相手側にも同期
- 状態同期と演出同期は分離したまま維持
