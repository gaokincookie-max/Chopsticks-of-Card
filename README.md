# 割り箸カード v101 虚無発動・変身演出修正版

## 停止原因

虚無の演出が、存在しない `sleep()` 関数を呼んでいた。
カード使用直後にJavaScriptエラーとなり、変身・終端処理・画面解除が実行されなかった。

## 修正

- 既存の共通待機関数 `delay()` を使用
- 演出レイヤーを `try/finally` で必ず解除
- 虚無の変身処理全体も `try/catch/finally` で保護
- 処理中のみ `state.animating` を有効化
- 成功時に4加護を確実に変身後へ置換
- オンラインでは置換後の状態を明示同期
- 失敗時はログと画面メッセージへ原因を表示

## 演出変更

暗い演出を減らし、次を追加。

- 白金の中心光
- 虹色のオーロラ
- 放射状の光線
- 二重の光輪
- 星粒
- 明るいジョーカー
- 発光する4スート
- 白・金・桃・青を中心にした変身演出
