# 割り箸カード v43 同期競合修正版

## 修正内容

- `activeNoDraw` などオンライン同期で必要な状態を常に安全に初期化する `ensureOnlineStateMaps()` を追加
- 通常の自動同期は「現在手番を持つ端末」だけが書き込むよう変更
- 相手ターン中の端末が古い状態をFirebaseへ書き戻す競合を防止
- リモート状態を適用する直前に、保留中のローカル同期タイマーをキャンセル
- 狙撃・祈祷・カード使用済みフラグなどが古い状態で巻き戻る問題を抑制
- Safariで発生していた `state.activeNoDraw[player]` の未定義参照に防御初期化を追加
- キャッシュ番号を v43 に更新

## テスト優先項目

1. ゲスト側で試合開始時に `activeNoDraw` エラーが出ないこと
2. 祈祷を使った同じターンに2回目のカードを使えないこと
3. 狙撃で選んだ相手の手の数値が両端末で一致すること
4. 乱射が捨て札だけで止まらず、対象選択からダメージまで完了すること
