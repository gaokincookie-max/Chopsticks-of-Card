# 割り箸カード v96 チュートリアル説明確認・空振り修正版

## 空振り

手動罠の選択画面で「空振り」を選んだだけではなく、
実際に空振りの効果が発動し、攻撃無効処理が完了した後に
茨の課題へ進むよう修正。

## OKボタン

操作を必要としない説明では、対戦画面上部に「OK」を表示。
内容を確認してOKを押すと次へ進む。

## 分けるの追加説明

- 分けたターンは攻撃できない
- 攻撃か分けるのどちらか一方
- 分ける前後で左右の合計本数は変わらない
- 分けた結果として片方を0にする形にはできない

## 加護・呪縛・罠の追加説明

- 加護は自分の手へ付ける
- 呪縛は相手の手へ付ける
- 加護と呪縛は名前が公開され、継続して残る
- 罠は自分の手へ裏向きで置き、相手には種類が見えない
- 罠は条件で発動し、多くは一度で捨て札へ行く
- 罠・加護・呪縛は付いている手が0になると消える
