# 割り箸カード v52 捨て札タップ選択PVP版

## 変更内容

- PVPの「落ち着ける」をタップ選択方式に変更
  - 落ち着けるを使う
  - 捨てるカードを手札からタップ
  - 2枚引く
- PVPの「補修」をタップ選択方式に変更
  - 補修を使う
  - 捨てるカードを手札からタップ
  - 0の手を1で復活してターン終了
- どちらもキャンセルボタンを追加
- 乱射のタップ選択方式も継続
- `game.js?v=52` / `style.css?v=52` でキャッシュ対策

## 注意

PVP側の対象選択・捨て札選択はだいぶCPU戦風に近づいています。
