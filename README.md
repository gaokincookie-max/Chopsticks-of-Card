# 割り箸カード v61 防弾チョッキ・超過の呪縛調整版

## 新カード「防弾チョッキ」

- コスト2
- 加護
- この手は「狙撃」「乱射」によって本数を加えられない
- 「ロジックアトリエ」は防げない
- 手が0になったら捨て札になる
- 貫通弾は従来通り罠だけを破壊するため、防弾チョッキは貫通弾の破壊対象外

### 防御時
- 「防弾チョッキ」が遠距離ダメージを防いだ専用ポップアップを表示
- オンラインでは `bulletproofBlocked` 演出イベントとして両者に同期
- 乱射が防がれても弾薬は捨て札になる
- 加速弾・特殊弾・貫通弾を乱射で捨てた場合、ダメージが防がれても捨てられた時効果は発動する

## 「超過の呪縛」弱体化

変更前:
- 5以上になったら余り計算せず0

変更後:
- 7以上になったら余り計算せず0

通常攻撃・カード効果など共通の本数正規化処理と、通常攻撃の個別判定を両方7以上へ変更。
