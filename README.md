# 割り箸カード v54 PVP演出強化版

## 変更内容

- PVPで普通のカードを使った時も、画面中央にカード名演出を表示
- 攻撃時に、攻撃してきた手と攻撃された手を強調表示
- 攻撃演出は相手側の画面にも反映されるように、Firestoreの試合状態に演出情報を保存
- 5以上になった時、いったん合計値を表示してから計算後の数値に戻す演出を追加
- 弾カードの捨てられた時効果演出も相手側に反映
- `game.js?v=54` / `style.css?v=54` でキャッシュ対策

## まだ調整余地

- CPU戦と完全に同じ演出ではありませんが、PVP側でも同じ方向性の演出を入れています。
