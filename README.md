# 割り箸カード v52 最後の旋律・フィナーレ版

## 追加カード

- 最後の旋律
  - コスト3 / 補助
  - このターン最初に成立した共鳴で発動し、その攻撃手を強制的に0にする。
  - 実際に0へ変化した場合だけ「フィナーレ」を手札に加える。
  - 発動後はそのターン中の待機効果を消費する。
- フィナーレ
  - コスト0 / 終端 / トークン / デッキ投入不可
  - 相手の0でない左右の手それぞれに、自分の左右の合計本数分を加える。
  - 使用後ターン終了。

## 特別演出

- フィナーレ専用の暗転 → 開放 → 強い明転演出を追加。
- 相手両手の数値変化を同時に強調。
- オンラインでは `match.fx` の専用 `finale` イベントで相手側にも同じ特殊演出を送る。
- 最後の旋律には専用演出を追加していない。通常のカード使用ポップアップのみ。
