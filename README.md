# 割り箸カード v30 加護・呪縛版

GitHub Pages でそのまま動かせる分割版です。

## v30の主な変更

- 新種類「加護」を追加
  - 力の加護
  - 守護
- 新種類「呪縛」を追加
  - 鈍重の呪縛
  - 露呈の呪縛
- 加護は自分の手の下に表向きで置く
- 呪縛は相手の手の下に表向きで置く
- 罠ゾーン2枠を共有する
- 加護・呪縛は罠ではないため、解除・看破・手繰り寄せ・貫通弾では対象にならない
- 手が0になったら、その手の下のカードはまとめて捨て札になる
