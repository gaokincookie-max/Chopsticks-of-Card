# 割り箸カード v57 PVP操作改善版

## 変更内容

- PVPで通常時に自分や相手の手をタップしても、無意味な選択強調が残らないように変更
- 手の強調は、攻撃・カード対象選択など実際に選択が必要な場面だけ表示
- PVPの「乱射」をCPU戦と同じ流れに変更
  1. 乱射を使用
  2. 弾薬として捨てる手札を選択
  3. 攻撃する相手の0でない手を選択
  4. 乱射を解決してターン終了
- 0の手を乱射対象にした場合は解決せず、別の手を選び直せる
- `game.js?v=57` / `style.css?v=57` でキャッシュ対策
