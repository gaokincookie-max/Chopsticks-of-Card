# 割り箸カード v93 チュートリアル専用対戦分離版

通常CPU戦と同じ画面・カード・攻撃・設置処理を使いながら、
CPU思考や通常ターン進行だけを完全に分離したチュートリアル専用セッションへ変更。

## チュートリアル中に停止する処理

- CPUの自動思考
- CPUのカード使用
- CPUの通常攻撃・分ける
- ターン終了後のCPUターン移行
- CPUの追加行動
- 通常勝敗画面
- 通常のターン開始処理

## 維持する処理

- 通常対戦と同じ盤面
- 手札とカード使用
- 攻撃演出と計算
- 分ける画面
- 罠・加護・呪縛の設置
- 手動罠確認
- 自動罠発動
- ログと捨て札

練習CPUは第4章の指定された攻撃だけを、チュートリアルの台本から実行する。
