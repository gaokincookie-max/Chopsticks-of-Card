# 割り箸カード v55 デッキ編集強化版

## 変更内容

- デッキ保存を6スロット化
  - あなた用 / CPU用で別々に保存
  - 各スロットにデッキ名を設定可能
  - 現在表示中のデッキをスロットへ保存・読込
- 「表示中を空にする」ボタンを追加
  - 保存スロットは削除せず、編集中デッキだけを空にする
- 生成カード「ロジックアトリエ」「フィナーレ」をデッキ一覧へ表示
  - デッキ投入不可として表示
  - 効果説明を詳細画面で確認可能
- 「一点狙い」「最後の旋律」から生成カード詳細を直接確認可能
- 共鳴カードから「共鳴とは？」の詳細説明を確認可能
- デッキコードをWBDECK2形式へ刷新
  - デッキに入っているカードだけを記録してコードを短縮
  - URL-safe Base64を採用
  - 空白・改行・引用符・ゼロ幅文字を除去して読込
  - 旧WBDECK1コードの読込にも対応
  - 読込エラー理由を詳しく表示
- `game.js?v=55` / `style.css?v=55`
