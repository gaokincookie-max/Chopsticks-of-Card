# 割り箸カード v86 旧カード強化版

## 強化内容

- 強打：コスト2 → 1
- 過加速：コスト3 → 2
  - 追加ドロー2ターン → 3ターン
  - 反動のドロー不可2ターンは維持
- 補修：終端を削除
  - 修復後も攻撃か分けるが可能
- 探り：効果を完全刷新
  - 自分か相手の山札の一番上を確認
  - カード名・コスト・種類・効果を専用ポップアップ表示
  - オンラインでは確認内容を相手へ公開しない
- 倹約令：コスト3 → 2
- 反撃：コスト3 → 2
