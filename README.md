# 割り箸カード v82 乱闘・予告状と充電制限の整理版

## 乱闘

乱闘は選ばれたカードそのものを使用するのではなく、
そのカードの効果だけをコピーして発動する。

そのため、充電カードの「このカードは1ターンに1度しか使用できない」制限について:

- すでに通常使用済みの充電カードでも、乱闘で効果を発動できる
- 乱闘で発動しても、そのカード名を使用済みとして記録しない
- 乱闘後に元の充電カードを通常使用することもできる
- 必要な充電消費や発動条件は効果内で通常どおり確認する

## 予告状

### 公開した宣言ターン

予告状で充電カードを公開した時点で、そのカードを使用した扱いにする。

- 同じターンに通常使用済みの充電カードは予告できない
- 予告した後、そのターン中に同名充電カードを通常使用できない
- 公開時点で1ターン1回の使用済み状態へ記録

### 次の自分ターンの発動

次ターンの予告効果は、カードそのものではなく効果だけを発動する。

- 次ターンの1ターン1回制限を確認しない
- 次ターンの使用済み状態へ記録しない
- 予告効果の後に、同名充電カードを通常使用できる
- 必要な充電消費や効果内条件は通常どおり確認する

## 例

充電Lv.10で雷撃を予告:
- 予告したターン: 雷撃は使用済み扱い
- 次の自分ターン: 予告された雷撃効果が発動
- その後、手札の別の雷撃を通常使用することも可能
