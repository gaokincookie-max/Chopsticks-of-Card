# 割り箸カード v41 共通状態同期基盤版

## 今回の変更

- v38由来のCPU戦本体を正本として維持
- PVP専用カード定義・PVP専用戦闘画面は作らない方針を継続
- オンライン戦のゲーム状態を host / guest の共通形式でFirebaseへ保存
- 手の本数、山札、手札、捨て札、罠・加護・呪縛、各種継続状態、ターン、ログを同じ同期経路に統合
- 相手から届いた状態を各端末の human / cpu 視点へ変換し、既存CPU戦画面へ復元
- ターンが渡された側で既存の startTurn() を使い、ターン開始ドロー・過加速などを処理
- CPU戦の CARD_LIBRARY と既存カード処理をそのまま使うため、カードをオンライン専用に再実装しない基盤を追加

## 現段階の重要な制限

- 手動罠、ねこだまし、マジックミラーなど「相手側が途中で発動するか選ぶ」処理は、次の段階でリモート入力待機を追加する必要があります。
- 現在は共通状態同期の第一段階です。全カードを最終的に同じ処理で動かすための土台です。
