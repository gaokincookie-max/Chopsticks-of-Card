# 割り箸カードゲーム v107 無節制調整版

追加カード：
- すり減る希望 / 仲間と共に
- ヒステリー / 愛で！
- 色褪せた信条 / 騎士の信条
- 無節制 / 黄金狂
- 裏切られた心 / 友情
- 空虚な心 / 満ちる心

虚無の発動時、手札・山札・捨て札にある変身前カードを変身後カードIDへ置換する。
これによりカード名、説明、効果、虹枠表示がすべて変化する。

変身後カードは生成専用で、デッキへ直接投入できない。
デッキ編集では変身前カードから変身後の詳細を確認できる。

すり減る希望は対象カードを山札へ戻し、山札をシャッフルする。

魔法少女テーマは試作中のため、ゲーム内のお知らせは追加していない。


## v105 試験実装
- 「魔法少女の詠唱」と「アルカナ・スレイブ！！」を追加。

- 変身後の魔法少女加護がある場合は詠唱を省略して発動。
- 試験テーマのためゲーム内お知らせには未掲載。

## v107 試験調整

- 「無節制」に、使用した次の自分ターンはカードを使用できない代償を追加。
- 代償ターン開始時に専用ポップアップを表示。
- カード使用禁止中も、通常攻撃・分ける・パスは可能。
- 過充電などの行動不能状態とは別の専用状態として管理。
- CPU戦・フレンド戦の同期、再戦時の初期化に対応。
- 魔法少女テーマは試験実装中のため、ゲーム内のお知らせには未掲載。
