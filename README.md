# 割り箸カード v55 CPU戦UI統合版

## 主な変更

- PVP対戦画面をCPU戦の盤面構成・見た目へ大幅に統一
- 相手を上、自分を下に置くモバイル表示と、PC時の左右配置をCPU戦に合わせて調整
- 手の表示をCPU戦風の大型パネル・本数表示・指アイコンへ変更
- 攻撃は「攻撃 → 自分の手 → 相手の手」の直接タップ方式を維持
- 「分ける」を即実行式から、CPU戦と同じ「分け直しパネル → 決定」方式へ変更
- 解除ボタンを追加し、攻撃対象選択やカード対象選択をキャンセル可能に
- 山札/手札、対戦状況、手札、ログの配置をCPU戦に近づけて再構成
- PVPのFirebase同期処理・既存カード処理・v54の同期演出は維持
- `game.js?v=55` / `style.css?v=55` に更新してGitHub Pagesのキャッシュ対策

## GitHub Pages

以下の4ファイルを同じ階層に置いてください。

- `index.html`
- `style.css`
- `game.js`
- `README.md`

既存のファイルを置き換えてCommitすると更新できます。


## プレビューについて

- `index.html` は GitHub Pages 用で、`style.css` と `game.js` を同じ階層に置いて使います。
- `preview_standalone.html` は CSS / JavaScript を内蔵した単体プレビュー用です。ChatGPT等で1ファイルだけ開く場合はこちらを使えます。

## v55 修正

- オンライン対戦開始時、PVPデッキが無効でもデッキ編集画面へ自動遷移しないよう変更。
- 無効な場合はエラーメッセージを表示し、オンラインメニューに留まる。
