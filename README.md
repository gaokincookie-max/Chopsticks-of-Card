# 割り箸カード v83 オンライン光速回路・所有状態修正版

## 調査結果

host / guest の変換自体は正しかったため、
相手が光速回路を使っただけで直接自分の使用済みになる単純な mapping ミスではなかった。

問題は、オンラインで双方が次の内容を含む「両者分の全状態」を送信していたこと。

- 自分側の状態
- 相手側として保持しているコピー
- 光速回路の一試合一度状態
- 反動予約
- 今ターン使用済み充電カード

相手が古いコピーを持ったまま送信すると、
自分専用の状態が古い値で上書きされる競合が起こり得た。

## 修正

各オンラインスナップショットへ `publisherSide` を追加。

所有者専用状態は、所有者本人が公開したスナップショットからだけ更新する。

所有者専用:
- 光速回路の一試合一度使用済み
- 過充電・光速回路の反動予約
- 反動の原因
- 今ターン使用済みの充電カード

例:
- host の光速回路使用済みはhostの送信だけが更新
- guest の光速回路使用済みはguestの送信だけが更新
- 相手の古いキャッシュでは上書きされない

手の本数・罠・手札など、相手の攻撃で変化する通常戦闘状態は従来どおり共有する。

## 表示

光速回路が使えない場合は、手札に理由を表示。

- `光速回路はこの試合で発動済み`
- `この充電カードは今ターン使用済み`

クリック時にも、
`光速回路はこの試合ですでに正常発動しています`
と表示する。
