# 割り箸カード v71 予告状・公開演出版

## 予告状の公開処理

予告状でカードを選んだ瞬間に、通常のカード使用ポップアップに近い形で、
選ばれたカードを両プレイヤーへ公開するよう変更。

### 表示内容
- 上部ラベル: `○○の予告状`
- カード名: `予告「カード名」`
- `次の自分のターン開始時に発動`
- 選んだカードの効果全文

### 色
通常使用カードとは区別しやすいよう、黄土色・金色系の専用配色。

### オンライン
- 宣言した側に即時表示
- Firebaseの専用FX `advanceNoticeReveal` で相手側にも同期
- カードIDを明示的に送るため、相手側にも同じカード名・効果が表示される
- ポップアップ終了後、選択カードを捨て札にして予約状態へ登録

### 継続仕様
- 選んだカードは宣言時に捨て札へ置く
- 次の自分ターン開始時に効果だけ発動
- 使用条件とコストは再確認しない
