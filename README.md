# 割り箸カード v60 PVP共通UI基盤版

## 今回の目的

PVPを「CPU戦に似せた別UI」として修正し続ける方式をやめ、CPU戦の表示構造をPVP側でも直接使うための基盤を作り直しました。

## 主な変更

- PVPの左右の手をCPU戦と同じ `.hand / .fingers / .finger-icons / .trap-slots` 構造へ変更
- 加護・呪縛の表示もCPU戦と同じ罠スロット風の表示へ統一
- 対象選択時だけCPU戦と同じ選択可能表示を出す
- 通常時の無意味な手の選択強調は出さない
- カード使用ポップアップの同期判定を修正
  - 以前は「最後のログ」だけを見ていたため、ひらめき等では「1枚引いた」が最後になりポップアップが消えることがあった
  - v60では、その操作で新しく追加されたログから「○○を使用」を探すため、乱射以外でも中央表示される
- 乱射は v57 の「弾を選ぶ → 相手の手を選ぶ」を維持
- `style.css?v=60` / `game.js?v=60` に更新

## 重要

これは共通化の第一段階です。PVP内部のカード処理はまだ `friendUseCardAction()` 系が残っており、CPU戦の `CARD_LIBRARY / playCard()` と完全に同一の対戦エンジンにはまだなっていません。
今回の版では、まずUI構造と演出同期をCPU戦側へ寄せ、今後カード処理・罠・加護・呪縛を順に共通化しやすい形へ整理しています。
