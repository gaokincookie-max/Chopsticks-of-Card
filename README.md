# 割り箸カード v95 チュートリアル設置・盤面修正版

## 設置カード

罠・加護・呪縛は次の二段階で進む。

1. 手札のカードを選ぶ
2. 設置する手を選ぶ

カードを選択して設置モードへ入った時点で、
チュートリアルも「手を選ぶ」段階へ進む。

実際に対象の手へ置いた後で、次の説明へ移る。

## 強打から軽打

強打で相手を0にした攻撃演出が終わってから、
軽打用の盤面を自分3・0、相手3・0へ再構築する。

演出の遅延によって古い0の盤面へ戻されないよう、
課題開始後にも固定盤面を再適用する。

## 章選択画面

- 長い説明文を枠内で折り返す
- カード幅のはみ出しを防止
- スマートフォンでは1列表示
- タイトル文字サイズを画面幅に合わせて調整
