# 割り箸カード v35 盤面干渉調整版

GitHub Pages でそのまま動かせる分割版です。

## v35の主な変更

- 新カード「すりかえ」を追加
  - コスト2 / 補助
  - 相手の加護・呪縛と自分の加護・呪縛を1枚ずつ選んで交換
- 「強行突破」を強化
  - このターン、自分の攻撃は相手側の罠・加護・呪縛の効果を受けない
  - 攻撃判定前だけでなく、攻撃判定後の相手罠も封じる
  - 相手側の守護などの防御効果も無視
- 「手繰り寄せ」を変更
  - 罠だけでなく、加護・呪縛も移動可能
- 「注目」「受け流し」のコストを3から2に変更
- デフォルトデッキに「すりかえ」を追加し、20枚・コスト40に調整
