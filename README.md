# 割り箸カード v51 共鳴テーマ第一弾

## 追加カード

- アレグロ（コスト2・補助）
  - このターン最初の共鳴で2枚ドロー。
- 共鳴調節（コスト2・加護）
  - この手は攻撃対象との本数差が1以下でも共鳴。
- クレッシェンド（コスト3・補助）
  - このターンの共鳴攻撃は攻撃力+2。
- 乱舞（コスト2・補助）
  - 次の攻撃はダメージなし。攻撃対象の本数を攻撃した手と同じにする。
- ラルゴ（コスト2・加護）
  - 共鳴攻撃の攻撃力+1。共鳴時に1枚ドロー。
- アンダンテ（コスト1・補助）
  - 自分の0でない手を選び、+1か-1。0にはできない。

## 共鳴ルール

通常は、攻撃開始時の「攻撃する手」と「攻撃対象の手」の本数が同じなら共鳴。
共鳴調節が攻撃する手に付いている場合、本数差1以下まで共鳴として扱う。
対象変更罠が発動した場合は、変更後の攻撃対象で共鳴を再判定する。
乱舞による本数合わせは、その攻撃自身の共鳴には数えない。
