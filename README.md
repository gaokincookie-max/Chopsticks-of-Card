# 割り箸カード v64 指令UI・カード見出し改善版

## 指令カード
- 黒背景を廃止して通常カードと同じ基本配色へ変更
- 指定攻撃は `指令：指定攻撃［R］` / `［L］`
- 対象指定は `指令：対象指定［R→L］` など、カード名だけでも指定を確認可能
- 指令本文を専用の簡潔表示へ変更
  - 指定・条件は青系
  - 達成報酬は緑系
  - 未達成効果は赤系
- 沈黙は「カード使用禁止」、再編成は「分ける」を強調

## CLEAR演出
- 約0.72秒から約1.35秒へ延長
- `CLEAR` / `CLEAR ×N` は維持
- 未達成は演出に含めない
- 後半に少し余韻を追加

## 全カード共通
- 補助 / 罠 / 弾 / 指令 / 使用不可 / デッキ編成不可などのラベルをカード名の横から下段へ移動
- ラベルがカード名の横幅を侵食し、不自然な位置で改行される問題を改善
- 手札とデッキ編集画面の両方へ適用
