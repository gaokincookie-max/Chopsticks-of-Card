# 割り箸カード v87 手札コンパクト表示設定版

## 新しい設定

設定画面へ「カード説明を省略する」を追加。

初期状態はOFF。

## OFF
従来どおり、手札に以下をすべて表示。

- カード名
- カード種類
- コスト
- 効果説明

## ON
手札カードをコンパクト表示。

- カード名
- カード種類
- コスト
- 「長押しで効果を表示」
- 通常の長い効果説明は非表示

## 長押し
カードを約0.55秒長押しすると、専用ポップアップで以下を表示。

- カード名
- 種類
- コスト
- 完全な効果説明

マウス・タッチ・ペンに対応。
長押し後に指を離した際、カードがそのまま使用されないようクリックを抑止。

## 保存
設定はlocalStorageへ保存。
同じブラウザでは次回起動時も維持される。
