# 割り箸カードゲーム v128

## 今回の変更

- 「控訴」「上告」で終端カードを無効化した場合、使用者へカード使用権を1回分返すように変更しました。
- 無効化されたカードと同名のカードは、そのターン中だけ再使用できません。
- 返却された使用権では、別名のカードを通常どおり使用できます。
- 通常使用権だけでなく、黄金狂などの追加カード使用権を消費していた場合も、使用前の残り回数へ戻します。
- 「上告」でも同名カード再使用禁止が正しく付くように修正しました。
- 天秤テーマは引き続き試験実装のため、ゲーム内のお知らせには掲載していません。

## 起動方法

`index.html` をブラウザで開いてください。GitHub Pages / Netlify Dropでも利用できます。
