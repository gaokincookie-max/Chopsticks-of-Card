# 割り箸カード

GitHub Pages でそのまま動かせる分割版です。

## ファイル構成

- `index.html`：画面の骨組み
- `style.css`：見た目
- `game.js`：ゲーム処理・カード定義・CPU処理

## GitHub Pages に置く方法

1. リポジトリを作る
2. このフォルダ内の `index.html`, `style.css`, `game.js` をアップロードする
3. Settings → Pages から GitHub Pages を有効化する
4. 表示されたURLを開く
