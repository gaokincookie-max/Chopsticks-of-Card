# 割り箸カード v75 空間切断・手選択修正版

## 原因

充電Lv.5～9で空間切断を使った場合、`dimensionalSlashSacrifice` モードには入っていたが、
盤面の左右の手をクリックした時にそのモードを処理する分岐がなかった。

そのため、手を選んでも通常の攻撃選択として扱われ、
手を0にする処理と空間切断の後続効果が発動していなかった。

## 修正後

充電Lv.5～9で空間切断を使用:

1. 自分の左右の手から、0でない手を選択
2. 選んだ手を0にする
3. その手の罠・加護・呪縛を解除
4. 充電を5消費
5. このターンの与える本数+1
6. 通常攻撃を2回まで可能にする

手を実際に0にできた場合だけ、後続効果が発動する。

## 選択制限

- 相手の手は選べない
- すでに0の手は選べない
- 0の手を選ぼうとした場合は選択モードを維持
- オンラインでは選択結果を状態同期
