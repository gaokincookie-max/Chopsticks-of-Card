# 割り箸カード v92 チュートリアル通常対戦UI統合版

## 変更点

v90/v91のチュートリアル専用簡易盤面を、通常のCPU戦画面へ統合。

章選択画面だけは専用画面を使用し、章開始後は以下をすべて通常対戦と共有する。

- 左右の手
- 攻撃する手と対象の選択
- 攻撃計算と演出
- 分けるボタン、左右の選択欄、決定ボタン
- 実際の手札カード
- 罠・加護・呪縛の設置
- 手動罠の発動確認
- 自動罠の処理
- 捨て札とログ

チュートリアル側は固定盤面、固定手札、正しい操作の強調、章進行のみを担当する。

1. 攻撃を使おう
2. 分けるを使おう
3. カードの使い方
4. 罠を使おう
5. 加護と呪縛
