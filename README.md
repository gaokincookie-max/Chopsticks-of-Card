# 割り箸カード v59 盤面視認性・加護呪縛ヘルプ・捨て札効果演出版

## 変更内容

### 設置カードの色分け
- 看破・露呈などで公開された罠: 紫
- 加護: 緑
- 呪縛: 赤紫
- 盤面表示を `罠｜名前` / `加護｜名前` / `呪縛｜名前` に変更
- `revealed-info` の黄色が加護・呪縛の色を上書きしていたCSS競合を解消

### ヘルプ
- 「加護・呪縛」タブを追加
- 加護の設置先、表向き情報、継続効果、0になった時の処理を説明
- 呪縛の設置先、表向き情報、継続制限、0になった時の処理を説明
- 加護カード一覧・呪縛カード一覧を種別ごとに表示
- 罠との違いを説明

### 弾の捨て札効果演出
- 加速弾
- 特殊弾
- 貫通弾

上記カードの「捨てられた時効果」が発動した瞬間、専用のカード効果ポップアップを表示。
オンライン対戦では `match.fx` の `discardEffect` イベントとして相手側にも同期。
状態同期とは分離しているため、ポップアップ表示だけで盤面状態を変更しない。
