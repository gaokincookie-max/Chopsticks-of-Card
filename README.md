# 割り箸カード v40 簡易カードPL対戦版

GitHub Pages でそのまま動かせる分割版です。

## v40の主な変更

- PL vs PLで、簡易カードあり対戦を追加
- 試合開始後はロビーではなく専用の対戦画面へ移動
- Firestoreで以下を同期
  - 左右の手
  - ターン
  - 山札
  - 手札
  - 捨て札
  - 使用カード
  - 簡易ログ
- 分けるで片手0にできていた問題を修正
