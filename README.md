# 割り箸カード v42 共通同期安定化版

## 修正内容

- オンライン試合開始時に `pendingNoDraw / activeNoDraw` が未初期化で `Cannot read properties of undefined (reading 'human')` になる問題を修正
- 過加速の反動状態もオンライン同期対象に追加
- `render()` 中の同期予約に安全判定を追加し、乱射など複数段階カードの途中状態を送信しにくく修正
- カード効果解決中は状態同期を保留し、効果処理完了後に同期するよう修正
- オンライン戦開始時にターン関連の状態コンテナを明示的に初期化
- 古い/欠損した同期データを受けても `startTurn()` が落ちにくいよう防御処理を追加

## 狙い

乱射の「カードを捨てただけの途中状態」が相手へ同期される問題や、古い `cardActionUsed` が巻き戻って次ターンにも残る問題を抑えるための基盤修正です。
