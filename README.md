# 割り箸カード v78 雷撃・光速回路調整版

## 雷撃

### 変更前
- 充電3消費
- 使用前の充電3ごとに攻撃+1

### 変更後
- 充電5消費
- 使用前の充電4ごとに、次の攻撃+1

攻撃増加:
- 充電0～3: 不発または+0
- 充電4～7: +1
- 充電8～10: +2

ただし雷撃は充電5が必要なので、実際には:
- 充電5～7: +1
- 充電8～10: +2

使用前が充電Lv.10なら、
次の攻撃で超過計算前の合計が5以上になった時、あまりを計算せず0にする。

使用後:
- Lv.5 → 0
- Lv.8 → 3
- Lv.10 → 5

## 光速回路

ほかの充電カードと同じ不発方式へ変更。

### 充電Lv.0～9
- カードは使用され、捨て札へ
- 効果は不発
- 一試合一度の使用済みにはならない
- 充電カード使い放題は始まらない
- 次ターン行動不能も付かない

### 充電Lv.10
- 正常発動
- 一試合一度の使用済みになる
- そのターン、充電カードを何枚でも使用可能
- 充電カードの終端を無視
- ターン終了時に充電0
- 次の自分ターンは行動不能

## 行動不能の修正

行動不能フラグをオンライン共有状態へ追加。

次の自分ターン:
1. 通常の開始時ドロー
2. 予告状などの開始時効果
3. 行動不能を表示
4. 攻撃・分ける・カード使用を行わず自動ターン終了

CPU戦・オンライン対戦の両方で保存される。
