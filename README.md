# 割り箸カード v42 手選択UI改善版

GitHub Pages でそのまま動かせる分割版です。

## v42の主な変更

- PL vs PL簡易カード対戦で、手を直接タップして選べるUIを追加
- 自分の手をタップすると攻撃元/自分対象に設定
- 相手の手をタップすると攻撃先/相手対象に設定
- 選択中の自分の手は青系、相手の手は赤系で表示
- `adjust` などカードIDがそのまま表示される問題を修正
- GitHub Pagesのキャッシュ対策として `game.js?v=42` / `style.css?v=42` を使用
