# 割り箸カード v41 カード追加UI調整版

GitHub Pages でそのまま動かせる分割版です。

## v41の主な変更

- PL vs PL簡易カード対戦の対応カードを追加
- 対戦画面を少し元のUIに近づける調整
- 自分/現在の番/カード効果の選択欄を表示
- 狙撃などが選択中の相手の手を参照するように変更
