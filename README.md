# 割り箸カード v36 初手防御調整版

GitHub Pages でそのまま動かせる分割版です。

## v36の主な変更

- 新カード「超過の呪縛」を追加
  - コスト3 / 呪縛
  - この手は5以上になったら余り計算をせず0になる
- 新カード「ねこだまし」を追加
  - コスト2 / 補助
  - 通常使用時は1枚ドロー
  - 自分の初ターンが来る前に攻撃されるとき、手札から捨てて攻撃を無効化
  - 乱射などの終端攻撃カードも無効化可能
- 「衰弱の呪縛」を待機1に変更
  - 置かれた後、持ち主のターン終了を1回待機
  - その次のターン終了時から-1が発動
- デフォルトデッキに「ねこだまし」「超過の呪縛」を追加
- デフォルトデッキは20枚・コスト40に調整
