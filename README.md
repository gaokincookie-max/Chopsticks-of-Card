# 割り箸カード v44 オンライン割り込み同期版

## 今回の変更

- v43の同期競合修正を維持
- オンライン対戦で、相手本人の判断を待つ「割り込み同期」を追加
- 手動罠の発動確認を相手端末に表示
- ねこだましの使用確認を相手端末に表示
- マジックミラーの使用確認を相手端末に表示
- 判断中はゲーム状態の通常同期を止め、回答後に元の処理を再開
- CPU戦では従来どおりCPU判断を使用

## テストしてほしい項目

1. ねこだましを持つ側が、初ターン前の攻撃時に発動確認を受け取れるか
2. 受け流し・空振り・踏み止まり・反撃・スワンプマン等の手動罠で、守備側本人に選択画面が出るか
3. 発動しないを選んだ場合も攻撃が正常に続くか
4. マジックミラーで呪縛設置時に相手側へ確認が出るか
5. 判断中に状態が巻き戻ったり、二重に処理されないか

## 注意

- 実際の2端末Firebase通信はブラウザ環境でのテストが必要です。
- 自動罠は従来どおりゲーム処理側で自動発動します。
