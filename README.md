# 割り箸カード v85 お知らせセンター実装版

## ホーム画面
ホームメニューに「お知らせ」ボタンを追加。

新しい更新をまだ開いていない場合は、
ボタン右上に `NEW` バッジを表示する。

一度お知らせを開くと、ブラウザのlocalStorageへ既読状態を保存する。
アカウントやログインは不要。

## お知らせ一覧
新しい更新から順に表示。

分類:
- NEW CARD
- BALANCE
- FIX
- SYSTEM

分類ボタンで絞り込み可能。
各更新を押すと詳細が開閉する。

## 大型アップデート告知
v85を初めて開いた時に一度だけ、
新テーマ「光速に灼かれた紫電」の全画面告知を表示。

表示内容:
- NEW THEME RELEASE
- 光速に灼かれた紫電
- 12 NEW CARDS
- OVERCLOCK
- CHARGE SYSTEM

「更新内容を見る」でお知らせ一覧を開く。
「閉じる」でホーム画面へ戻る。

同じブラウザでは一度閉じると自動表示されない。
お知らせ一覧からはいつでも内容を確認できる。

## 更新データ
`game.js`内の `UPDATE_NEWS` の先頭へ項目を追加するだけで、
今後のアップデート履歴を増やせる。
