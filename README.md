# 割り箸カード v73 空間切断・充電条件修正版

## 空間切断

### 充電0～4
- カードは使用され、捨て札へ移動
- 効果は完全に不発
- 充電は消費しない
- 自分の手を0にしない
- 攻撃+1と2回攻撃も得ない

### 充電5～9
- 充電を5消費
- 自分の手を1つ選んで0にする
- このターン与える本数+1
- 通常攻撃を2回まで行える

### 充電10
- 充電を5消費し、Lv.5になる
- 自分の手を0にするデメリットなし
- このターン与える本数+1
- 通常攻撃を2回まで行える

## 乱闘との関係
乱闘で空間切断が選ばれた場合も充電条件を確認する。

- 充電5未満なら不発
- 充電5～9なら手を0にして発動
- 充電10ならデメリットなしで発動
- 空間切断本体は乱闘の仕様により手札に残る
- そのターンの空間切断使用済み状態は記録される
