# 割り箸カード v66 凶弾・指令連動版

## 指令と凶弾

「凶弾」による自分の手への攻撃も、指令の攻撃履歴として記録するよう変更。

### 指令：指定攻撃
- 指定された手で攻撃すれば達成
- 通常攻撃だけでなく「凶弾」も対象
- 例: 指令：指定攻撃［R］を持ち、右手で凶弾を行う → 達成

### 指令：対象指定
- 指定された攻撃手 → 指定された対象手の組み合わせで攻撃すれば達成
- 通常攻撃 / 凶弾の両方が対象
- 例: 指令：対象指定［R→L］を持ち、右手から左手へ凶弾 → 達成
- 通常攻撃で右手から相手左手を攻撃した場合も同じく達成

## UI
- 指令の説明から「相手を通常攻撃」という相手限定表現を削除
- 「手で攻撃」「R→L」のような簡潔表示を維持
- 指定攻撃・対象指定には「通常攻撃 / 凶弾で達成可能」と表示
