# 割り箸カード v69 乱闘・予告状・決闘高潮

## 乱闘
- コスト2 / 補助
- 手札からランダムに1枚を選び、条件とコストを無視して効果だけを発動
- 選ばれた元カードは手札に残る
- 終端・生成カードも候補
- 除外: 乱闘、指令、ロジックアトリエ
- 対象選択が必要な効果は通常どおり選択
- 終端カードが選ばれた場合は終端も有効

## 予告状
- コスト2 / 補助
- 宣言時点で通常の使用条件を満たすカードだけ選択可能
- 選んだカードを相手に公開し、その場で捨て札へ置く
- 次の自分のターン開始時、条件とコストを再確認せず効果だけを発動
- 捨て札から回収されても予約効果は消えない
- 終端カードも予告可能
- 除外: 予告状、指令、ロジックアトリエ

## 決闘高潮
- コスト3 / 加護
- 加護が付いた手自身の通常攻撃だけを記録
- 同じ対象を攻撃するとLv上昇、別対象ならLv1
- 最大Lv5
- 反対の自分の手で攻撃してもレベルは変化しない
- 対象変更罠が発動した場合、最終的な攻撃対象を記録

### レベル効果
- Lv0-1: 攻撃+0 / 軽減0
- Lv2: 攻撃+1 / 軽減0
- Lv3: 攻撃+1 / 軽減1
- Lv4: 攻撃+2 / 軽減1
- Lv5: 攻撃+2 / 軽減2
- 軽減後は最低1本

### 表示
- 盤面: `✦ 決闘高潮 Lv.N` のみ
- 詳細: 記録対象、現在の攻撃増加、被攻撃軽減、次のレベル条件
