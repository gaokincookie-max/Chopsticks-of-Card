# 割り箸カード v90 初心者チュートリアル版

## 全5章

1. 攻撃を使おう
   - 自分の手を選んで相手を攻撃
   - 5になったら0
   - 6以上は5を引いた余り

2. 分けるを使おう
   - 自分2・0、相手3・2
   - 2・0を1・1へ分けて敗北を回避
   - 分けたターンは攻撃できない

3. カードの使い方
   - ひらめきで1枚引く
   - 自分1・0、相手3・0で強打
   - 自分3・0、相手3・0で軽打
   - 終端カード「パス」でターン終了

4. 罠を使おう
   - 空振りを置き、発動するか手動選択
   - 茨を置き、自動発動を体験

5. 加護と呪縛
   - 力の加護を自分の手へ設置
   - 鈍重の呪縛を相手の手へ設置
   - 罠との違いと、手が0になると消えることを説明
