# 割り箸カード v43 CPU風UI演出カード制限版

GitHub Pages でそのまま動かせる分割版です。

## v43の主な変更

- PL vs PL簡易カード対戦の手の増減に演出を追加
  - 増えた手は軽く大きくなる
  - 減った手は軽く縮む
- 1ターンにカードを複数枚使えてしまうバグを修正
  - 各ターン、カード使用は1回まで
  - 次の自分ターン開始時にカード使用権が戻る
- カード使用済みの場合、手札カードを押せないように表示
- 状態メッセージに「カード使用済み」を表示
- GitHub Pagesのキャッシュ対策として `game.js?v=43` / `style.css?v=43` を使用

## 注意

PL vs PLはまだ簡易カード対戦です。CPU戦と完全同一の処理ではなく、Firestore同期用の別処理です。
