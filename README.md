# 割り箸カード v28 CPU強化版

GitHub Pages でそのまま動かせる分割版です。

## ファイル構成

- `index.html`：画面の骨組み
- `style.css`：見た目
- `game.js`：ゲーム処理・カード定義・CPU処理

## v28の主な変更

- CPU難易度を追加
  - やさしめ
  - 標準
  - 強め
- CPUが行動候補を点数化して選ぶように変更
- CPUデッキの傾向を見て、罠・弾丸系カードを少し優先するように変更
- 勝てる攻撃、相手の手を0にできる攻撃、危険な手の分け方などを優先

## GitHub Pages に置く方法

このフォルダ内の `index.html`, `style.css`, `game.js`, `README.md` をリポジトリにアップロードしてください。
