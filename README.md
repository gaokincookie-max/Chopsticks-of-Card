# 割り箸カード v58 デッキコード・スロット名改善版

## 変更内容

- WBDECK1 / WBDECK2 の接頭辞を大文字小文字を区別せず読み込むよう修正
  - `WBDECK2:`
  - `wbdeck2:`
  - `WbDeCk2:`
  などを同じコードとして扱う
- Base64URL 部分は大小文字を変更せず、そのまま復号する
- 保存スロットの名前をスロット選択欄へ反映
  - 例: `スロット1｜共鳴デッキ`
  - 空きは `スロット2｜空き`
- あなた用 / CPU用で、それぞれ対応する保存スロット名を表示
- `style.css?v=58` / `game.js?v=58` に更新

- 20枚ちょうど
