# 割り箸カード v63 指令テーマ「都市は命令する」

## 新テーマ：指令

指令カードは使用できず、手札にある状態で自分のターンを終了すると照合される。
達成した指令だけ短い `CLEAR` 演出を表示。
複数達成時は `CLEAR ×2` などと表示。
未達成は演出に含めず、各指令の成否はログへ残す。

### 指令：指定攻撃
- 引いた時に右手 / 左手をランダム指定
- 指定手で相手を通常攻撃すると達成
- 達成: 1枚引く
- 未達成: 手札をランダムに1枚捨てる

### 指令：対象指定
- 引いた時に自分の手と相手の手をランダム指定
- 指定組み合わせで通常攻撃すると達成
- 達成: 2枚引く
- 未達成: 指定された自分の手に1本加える
- 0の手など達成不能な指定も再抽選しない

### 指令：沈黙
- 手札からカードを使用せずターン終了すると達成
- 達成: 2枚引く
- 未達成: 次の自分ターンの通常ドローなし

### 指令：再編成
- 分けるを行うと達成
- 達成: 次の自分ターン開始時ドロー+1
- 未達成: 多い方の手に1本加える。同数ならランダム

## 指令サポート

- 指令の意味
- 循環する都市
- 指令の加護
- 意志の剣
- 都市の意志

## オンライン
- CLEAR演出をFXとして相手にも同期
- 指令状態・達成数・予約効果は通常の共通状態同期に含まれる
